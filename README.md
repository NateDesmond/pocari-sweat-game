# Pocari Sweat — Ion Catch 🥤

A single-file browser game and a history presentation about Pocari Sweat.

- **`index.html`** — *Ion Catch*, a hydration-catcher arcade game (WebAudio sound, speed control, mobile-ready).
- **`presentation.html`** — interactive 11-slide deck on the history & creation of Pocari Sweat.
- **`build_pptx.py`** — generates `Pocari_Sweat_History.pptx` (PowerPoint export).
